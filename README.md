# [1]WRITE BOARD의 경우 제목은 80자 이내, 내용은 500자 이내이나, 한글의 경우 바이트가 수가 상이하여 한글과 혼용될 경우 짦아질 수 있음을 감안해주십시요.
# 글작성 완료 이후 /w 를 붙이실 경우 자동으로 올려집니다. 다만 500자가 넘을 경우 임의로 500자까지만 저장이 됨으로 유의하십시요.
# [2]READ BOARD의 경우 보드 내 글을 모두 읽을 수 있습니다. choice 에 쓰여진 번호를 입력할 경우 글을 읽을 수 있습니다.
# 텍스트 넘버, 작성자, 제목으로 있으며 제목 옆 [숫자]의 경우 리플 갯수를 알려줍니다.
# 다음 목록을 읽기 위해서는 0번을 눌러 엔터쳐주시면 넘어가게 됩니다.
# 상세 읽기를 위해 choice 넘버를 고를 경우, 글을 자세히 읽을 수 있습니다. 
# 리플을 작성하고 싶을 때는 /r 을 눌러 보내신 뒤 리플을 작성할수 있습니다.
# 작성자 본인 및 admin에 한정하여 글을 삭제할수 있는 부분이 제목에서 보입니다.
# [3] FIND BOARD의 경우 제목 검색이 가능합니다. 검색 시 해당 텍스트와 맞는 제목으로 출력해줍니다. 그외는 게시판 읽기와 동일합니다. 
# BOARD 쪽에서 나오시면 [2]DATA BOARD 가 있습니다. 2번을 눌러 해당보드로 들어가시면
# DOWNLOAD DATA /UPLOAD DATA/FIND DATA/DELETE DATA 가 나와 있습니다.
# [1]을 눌러 DOWNLOAD DATA 로 들어가시면 숫자를 눌러 다운로드가 가능하도록 되어 있습니다. 해당 다운로드에 올려진 파일은 서버 폴더 내 cloud 라는 곳이며
# 다운로드 요청 시 client 에 downlads 파일로 전송됩니다.
# [2]를 눌러 UPLOAD DATA 로 들어가시면 클라이언트 내 uploads 라는 폴더 내에 있는 파일을 선택하여 서버 내 cloud 로 업로드 시킬 수 있습니다.
# UPLOAD 를 누른 뒤 뒤로 돌아가 다시 DOWNLOAD 로 가시면 해당 파일이 올려져 있는 것을 확인하실 수 있습니다. 
# [3] FIND DATA의 경우 누르게 될 경우, [1] 다운로드 내에 있던 자료를 검색할 수 있게 되어있습니다. 검색 이후는 동일하게 다운로드가 가능합니다.
# [4] DELETE DATA의 경우 admin 만 권한을 갖고 있으며, 해당 권한을 통해 접속 삭제가 가능합니다. admin 으로 접속 이후 동일한 경로로 접속 하실 경우, choice 를 통해
# 파일을 삭제할 수 있습니다.
